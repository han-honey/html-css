html 기본(필수) 태그들

 <!DOCTYPE html>  ------ 웹문서의 버전을 선언 (브라우저에게)
 <html>   -------------- 웹문서의 시작
  <head>  -------------- 머리말의 시작 (웹문서가 제대로 보이게)
   <title>    </title> --- 제목 (브라우저 탭)
  </head> -------------- 머리말의 끝
  <body>  -------------- 본문의 시작 (사용자에게 보여질 컨텐츠)
 
  </body>  -------------- 본문의 끝
 </html>  -------------- 웹문서의 끝

text와 관련된 태그들

bold bold - 볼드체
strong strong - 중요하다는 의미에서 볼드체
italic italic - 이텔릭체
em em - 강조한다는 의미에서 이텔릭체
small small - 글자 크기를 줄임
sub sub - 아랫첨자 (ex) H2O, CO2, O3
sup sup - 윗첨자 (ex) 200km2, 30cm3
mark mark - 마크펜 효과

시작과 종료태그는 서로 감싸듯이
1. mark i
2. mark i
3. mark i
4. mark i

> = &gt;

  = &nbsp; (문자 하나만큼의 높이를 유지하는) 공백

♘ = &#9816;
