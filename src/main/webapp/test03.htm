List - 목록

목록의 종류

  • 순서가 없는 목록 - <ul>
  • 순서가 있는 목록 - <ol>
  • 정의 목록 - <dl>

ul - 순서없는 목록

문제. 순서없는 목록 만들기

  • coffee
  • tea
  • milk
  • html
  • css
  • js

ol - 순서 있는 목록

  1. html
  2. css
  3. js

문제. 순서 있는 목록 만들기

  1. red
  2. green
  3. blue

ol의 목록앞의 모양 바꾸기 - type="1/a/A/i/I"

ol type="1"
  1. web
  2. java
  3. jsp
  4. DB
  5. job
ol type="a"
  1. web
  2. java
  3. jsp
  4. DB
  5. job
ol type="A"
  1. web
  2. java
  3. jsp
  4. DB
  5. job
ol type="i"
  1. web
  2. java
  3. jsp
  4. DB
  5. job
ol type="I"
  1. web
  2. java
  3. jsp
  4. DB
  5. job

ol의 시작번호 바꾸기 - start="n"

  1. web
  2. java
  3. jsp
  4. DB
  5. job
  1. web (ol start="7" type="i")
  2. java
  3. jsp
  4. DB
  5. job

하위목록 (목록 안의 목록) 만들기

  • coffee
    • coffee-1
    • coffee-2
    • coffee-3
  • tea
    • green tea
    • red tea
  • milk
    • hot
    • cold

문제. 하위목록 만들기
