html5에서 추가된 input type값들

input type="time" 날짜 또는 시간 (오전/오후 시 분)

input type="date" 날짜 또는 시간 (연월일)

input type="datetime-local" 날짜 또는 시간 (오전/오후 시 분 연월일)

input type="week" 날짜 또는 시간 (연주)

input type="month" 날짜 또는 시간 (연월)

input type="number" 숫자값 입력시

  2. min="0" 최소
  3. max="100" 최대
  4. step="12" 증감간격

input type="range" 범위내의 값을 선택

input type="tel" 연락처(전화번호) 입력시

모바일브라우저에서 가상키패드가 숫자형식으로 바뀌거나 전화걸기 가능

input type="email" 연락처(이메일) 입력시

브라우저에서 email형식이 맞는지 체크해 줌(@유무, @이후)

input type="url" 연락처(웹주소) 입력시

브라우저에서 url형식이 맞는지 체크해 줌(프로코콜 유무)

input type="color" 색상값을 선택

input type="file" 파일 전송용

input type="text"의 속성들

  • size="n" n개의 글자 입력 가능한 상자 길이
  • maxlength="n" 최대 입력 글자 n개
  • minlength="n" 최소 입력 글자 n개

html5에서 추가된 input type="text"의 속성들

  • placeholder="..." 설명문이나 예시 등
  • autofocus(autofocus="autofocus") 커서가 해당 요소에 입력대기 상태로 있음
  • autocomplete="on/off" 자동완성 켜거나/끔
  • required(required="required") 필수입력요소로 지정
  • readonly(readonly="readonly") 읽기만 가능한 요소로 지정.(제출 할 때 전송됨)
  • disabled(disabled="disabled") 사용불가 요소로 지정.(제출 할 때 전송되지 않음)
