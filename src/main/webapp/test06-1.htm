a로 내부링크 걸기

 페이지 내에서 특정한 위치로 이동하기 위해서 주소로 활용할 id가 있어야 함.
            (id는 중복적용이 절대 불가하므로 주소로 활용.)
 
 따라서 
  1. 이동할 특정한 위치의 태그에 id를 추가
  2. a태그의 href속성값에 #id값을 지정

<a href="이동할 경로(#id)"> ... </a> (클릭시 아래로 이동)


<tag id="abc">

<tag id="bcd">

<a href="이동할 경로(#id)"> ... </a> (클릭시 위로 이동)
